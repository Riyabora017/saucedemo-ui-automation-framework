## SauceDemo UI Automation Framework

## Project Overview

This project is a UI Automation Testing Framework built for the SauceDemo web application using Selenium WebDriver, Java, and TestNG.
It follows the Page Object Model (POM) design pattern and includes reporting, listeners, and reusable components for scalable automation.

##  Application Under Test
## Website: https://www.saucedemo.com/

 ## Tech Stack
**Java**
**Selenium WebDriver**
**TestNG**
**Maven**
**Page Object Model (POM)**
**Extent Reports**
**TestNG Listeners**
**JSON (Test Data Handling)**

## Reporting

After execution, Extent Reports will be generated in the /reports directory.

Open the HTML file in a browser to view detailed test execution results.
